Câu 1:
    a. git init: khởi tạo thư mục quản lý bởi git
    b. git add <file_name>: đưa 1 file vào vùng staging
    c. git add .: đưa all file vào cùng staging
    d. git commit -m “message”: commit file lên Repository

    d. add file 1 lên Staging
    e. commit file 1 lên Repository
