1. 
    - git commit --amend: vào chế độ edit commit message
    - git commit --amend -m”<message>”: thay đổi commit message mà không cần vào chế độ edit.
    - git restore --staged <file>: đưa 1 file từ Staging về working directory
    - git reset HEAD~1 : đưa 1 commit từ repository về working directory 
    (sửa bài: Lệnh này hoàn tác commit gần nhất, đưa các thay đổi trong commit đó trở lại khu vực stagingchứ không trực tiếp về working directory)
